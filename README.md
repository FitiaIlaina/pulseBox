# PulseBox – Lecteur multimedia

PulseBox est une application mobile de lecture de musique moderne et intuitive, conçue pour offrir une expérience fluide et immersive.  
Elle permet aux utilisateurs d’écouter et de gérer leurs musiques directement depuis leur smartphone.

Ce projet est développé avec **React Native** et **TypeScript**, en mettant l’accent sur la performance, la clarté du code et la maintenabilité.


##  Objectif du projet

- Créer un **lecteur de musique mobile fonctionnel**
- Mettre en pratique **React Native + TypeScript**
- Gérer la lecture audio et les fichiers multimédias

## Technologies utilisées

- **React Native** – Développement mobile cross-platform
- **TypeScript** – Typage fort et fiabilité du code
- **Expo** – Environnement de développement
- **Expo AV** – Gestion de la lecture audio
- **React Navigation** – Navigation entre les écrans

## Pour lancer l'application
npx expo start / npm start
